WhatsApp Tools - User Documentation

Getting Started

WhatsApp Tools is a Chrome extension that helps you extract phone numbers from WhatsApp Web and send bulk messages efficiently.

Number Extractor

How to Extract Phone Numbers

  1. Open WhatsApp Web in your browser
  2. Navigate to a chat or group containing phone numbers
  3. Click on the WhatsApp Tools extension icon
  4. Select the "Number Extractor" tab
  5. Click the "Extract" button
  6. The extension will display all phone numbers found on the page
  7. Use "Copy to Clipboard" to copy all numbers at once

Bulk Message Sender

How to Send Bulk Messages

  1. Click on the WhatsApp Tools extension icon
  2. Select the "Bulk Sender" tab
  3. Enter phone numbers (one per line) or use the "Transfer" button from extracted numbers
  4. Type your message in the message field
  5. Click "Start Sending" to begin the process
  6. The extension will open each chat and send your message automatically
Note: You can pause or stop the sending process at any time using the Pause/Stop buttons.

Message History

Managing Message History

  1. The extension automatically saves all successfully sent messages
  2. Use "Filter Already Contacted" to remove numbers you've messaged before
  3. Export your message history to Excel using the "Export to Excel" button
  4. Clear your message history with the "Clear History" button if needed

Troubleshooting

Common Issues

  • No numbers found: Make sure you're on a WhatsApp Web page containing phone numbers
  • Failed messages: Check the "Failed Numbers" tab to see which numbers couldn't be contacted
  • WhatsApp not loading: Ensure you're logged into WhatsApp Web before using the extension
